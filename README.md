# Cryo_LCVCO

Projeto de VCO (Voltage Controlled Oscillator) para 4.9 GHz em IHP 130nm.

| Parâmetro          | Valor           |
| ------------------ | --------------- |
| Tecnologia         | IHP 130 nm      |
| Frequência central | 4.9 GHz         |
| Faixa de sintonia  | 4.7–5.0 GHz     |
| Vctrl              | 0–3 V           |
| Alimentação        | 1.5 V           |
